1. CSS3线性渐变

语法：background:linear-gradient(direction, color-stop1, color-stop2,...);

完全支持该属性的第一个浏览器版本: IE10, Chrome 26.0, Firefox 16.0, Safari 6.1, Opera 12.1

注：IE9及之前不支持渐变。


线性渐变-从上到下(默认情况下)。

线性渐变-从左到右。

线性渐变-对角。

2. 使用角度

语法：background:linear-gradient(angle, color-stop1, color-stop2);

3. 使用多个颜色结点

4. 使用透明度

使用rgba()函数定义颜色结点

5. 重复的线性渐变

repeating-linear-gradient()用于重复性渐变

6. CSS3径向渐变

至少定义两种颜色结点。也可以指定渐变的中心、形状、大小。

默认情况下，渐变中心是center(中心点)，渐变形状是ellipse(椭圆形)，渐变大小是farthest-corner(到最远的角落)

语法：background:radial-gradient(center,shape size,start-color,...,last-color)


径向渐变-颜色结点均匀分布(默认情况下)

径向渐变-颜色结点不均匀分布

7.设置形状

shape参数定义了形状，可以是circle或ellipse。

8. 重复径向渐变

repeating-radial-gradient()用于重复性渐变
